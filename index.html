How to Play

You and the machine start with $10.
Each round, both choose an action secretly.
After 33 rounds, the richer player wins.
You can end the game early with Kill or End.
First to 3 wins takes the series.

Kill can only be executed by the poorer player. If it fails, they lose.
End ends the game, awarding the win to the richer player.

The machine learned by playing 100,000 games against itself.
